# Bootstrap 3

## Agenda

- 為何使用Bootstrap
    1. 開發人員與設計(美編)人員
    2. 什麼是Bootstrap
    3. 行動網頁
    4. 將網頁套入Bootstrap

## 開發人員與設計(美編)人員

- 不是每一個案子都有美編設計人員
- 開發人員盡心盡力將網站"做好"
- 但結果往往不是"那麼好"


## 什麼是Bootstrap

- 開放源始碼
- 專用來建置網頁
    - 用了一些css
    - 用了一些javascript
    - 模組化
    - 包含了80%以上的應用

## 行動網頁

- 行動裝置的重要性不下於桌上裝置
    - 低頭族 
    - 網頁設計重點放在行動裝置上
    - 你的網頁設計必需同時適用在行動裝置及桌上電腦
